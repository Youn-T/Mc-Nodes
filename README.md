# Mc Nodes - Minecraft Bedrock Mod Creator

A visual tool for creating Minecraft Bedrock Add-ons using a nodal system for scripting and forms for data-driven components.

## Features

- **Node Editor**: Visual scripting using React Flow.
- **Item Creator**: Create custom items with JSON generation.
- **Entity Creator**: (Coming Soon)

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React
- TypeScript
- Vite
- React Flow
- JSZip
